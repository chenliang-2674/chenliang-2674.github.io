将腾讯云上的centos系统重装为windows系统，并从本机向远程windows系统传送文件

之前在腾讯云上申请的是centos系统，现在想将centos系统重装为windows系统。

重装系统

登录进腾讯云，在下面的图中点击更多：

选择重装系统。点击后界面如下：

选择公共镜像，再选择需要装的windows系统，我装的是windows 2008版。如果想要更大的系统盘是需要花钱的，因为初次试，所以选择默认的免费的。输入登录密码，点击开始重装，即可重装系统。

从本机向远程windows系统传送文件

重装好系统后，点击登录，选择使用RDP文件登录：

下载RDP文件，打开，输入远程服务器IP地址：

因为我之前登录过，所以没有输入用户名和密码这个界面。

再选择本地资源，选择远程windows系统能访问的盘，我勾选了C盘和E盘。

再点击确定，就可以连接上远程桌面了。

打开远程桌面的文件管理器，可以查看公共的C盘和E盘，便可下载里面的文件了。

第二种方式可以直接复制本机文件，在远程桌面点击右键也可将文件粘贴过去。
第三种办法：使用FileZilla
我们需要在远程服务器中安装 FileZilla_Server，在本机安装 FileZilla 客户端。
