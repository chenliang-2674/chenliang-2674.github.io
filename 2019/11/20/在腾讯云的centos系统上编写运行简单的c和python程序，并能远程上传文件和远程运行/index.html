二 使用Xftp6将电脑上编写的程序文件上传到云服务器的上

首先下载安装Xftp6，可以选择安装路径，然后直接下一步，直到结束。
安装好后运行，会跳出两个窗口。在会话那个窗口上点击新建。

输入服务器IP地址，以及登录的用户名和密码，点击确定，再点击连接。

连接上以后左边为电脑文件，右边为远程服务器的文件，选中左边的需要上传的文件，直接拖到右边就上传了。

参考的Xftp6使用方法链接：
https://jingyan.baidu.com/article/1612d500f45b62e20e1eeee6.html
