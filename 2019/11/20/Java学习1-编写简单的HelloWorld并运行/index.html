Java学习1-编写简单的HelloWorld并运行

配置java编译环境

编写java 程序之前首先需要安装java的编译环境即JDK，从oracle官网下载JDK,下载链接：https://www.oracle.com/technetwork/java/javase/downloads/jdk12-downloads-5295953.html
下载好后，运行.exe文件，直至安装好。
安装好后先添加环境变量，将java.exe文件所在路径添加至环境变量中(右键我的电脑选择属性，再选择高级系统设置，进入选择高级里的环境变量，在系统变量中找到path,选中，点编辑，将路径添加进去）：

添加好环境变量后，打开cmd，输入”javac”，查看是否安装成功：

可以用”java -verison”查看版本：
